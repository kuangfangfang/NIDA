▸ How it works
01
Packet Capturing & Sniffing
Raw traffic flows are captured dynamically from active network adapter cards via Scapy. Packets are temporarily gathered in a local buffer stream.
02
Flow Extraction (CICIDS2017 Format)
Packets are grouped into bidirectional IP network flows based on their 5-tuple SrcIP, DstIP, SrcPort, DstPort, and Protocol. 78 statistical properties are calculated.
03
XGBoost Stage-1 Classification
Calculated features are passed to a Binary Classifier. The model calculates the likelihood that each individual traffic flow is malicious or safe.
04
XGBoost Stage-2 Attack Taxonomy
Malicious flows are routed into the Multiclass Classifier. It classifies the anomaly class (e.g. DDoS, Port Scan, Botnet) to outline precise actions.
