⚠️ Intrusion Detected
▸ How it works
01
You upload your traffic file
Got a CSV from CICFlowMeter? Perfect — we read it directly. Got a raw PCAP? No problem — we extract the flow features on the fly using scapy, no extra tools needed.
02
Stage 1 — Is this an attack?
The first model does one simple job: yes or no. It checks every network flow in your file. If everything looks clean — you're all good, we stop here. If something looks suspicious, we flag it and move on.
⬡ XGBoost binary classifier
03
Stage 2 — What kind of attack?
Only runs if Stage 1 found something. Takes the suspicious flows and figures out exactly what type of attack it is — then gives you plain-English advice on what to do right now.
⬡ XGBoost multi-class
