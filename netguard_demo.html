▸ How it works
01
Upload your traffic file
Got a CSV from CICFlowMeter? Perfect. Raw PCAP? We extract CICIDS2017 flow features on the fly using scapy — no extra tools needed.
02
Stage 1 — Is it an attack?
First model does one job: BENIGN or ATTACK. Checks every flow in your file. If all clear — you're done. Suspicious traffic gets flagged and passed straight to Stage 2. Trained with SMOTE balancing on 80% of CICIDS2017.
⬡ XGBoost binary classifier
03
Stage 2 — What kind?
Only runs if Stage 1 finds something. Identifies the exact attack category — trained only on attack-type records so it's laser-focused. Then gives you plain-English steps on what to do right now.
⬡ XGBoost multi-class
